# Decision-Trees-and-Random-Forests

## 📌 Project Overview

This project focuses on predicting the presence of heart disease using machine learning models such as **Decision Tree** and **Random Forest**.
The goal is to compare model performance, reduce overfitting, and interpret feature importance.

---

## 📊 Dataset

* **Name:** Heart Disease Dataset
* **Target Variable:** `target` (0 = No Disease, 1 = Disease)
* Contains medical attributes like age, cholesterol, blood pressure, etc.

---

## 🚀 Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

* Split data into training and testing sets
